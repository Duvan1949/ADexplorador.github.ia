Material de estudio Bootcamp Analisis de Datos Talento Tech

En esta pagina se detalla el material empleado y los recursos adquiridos durante las 159 horas de formación intensiva del Bootcamp de Análisis de Datos en el nivel Explorador. Este programa abarcó una amplia variedad de herramientas, técnicas y conocimientos esenciales para el análisis de datos, combinando teoría y práctica. A través de actividades guiadas, ejercicios prácticos y proyectos aplicados, se logró consolidar una base sólida para desempeñarse en el área de análisis de datos con enfoque exploratorio.
